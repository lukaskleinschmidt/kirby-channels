# Kirby UUID

*Working on it*

## Installation

### Download

Download and copy this repository to `/site/plugins/uuid`.

### Git submodule

```
git submodule add https://github.com/lukaskleinschmidt/kirby-uuid.git site/plugins/uuid
```

### Composer

```
composer require lukaskleinschmidt/kirby-uuid
```
